# Routing map

Use the smallest set that covers the decision.

| Need | Primary skill | Useful companion |
|---|---|---|
| Vague or multi-system problem | `founder-business-consultant` | `audit-business` |
| Idea, market, or demand proof | `validate-business-idea` | `founder-playbook-mom-test`, `founder-playbook-four-steps` |
| Root-cause business audit | `audit-business` | `reason-business-decision` |
| Formal tradeoff or high-uncertainty decision | `reason-business-decision` | relevant domain skill |
| Positioning and category | `founder-playbook-obviously-awesome` | `founder-playbook-blue-ocean`, `founder-playbook-crossing-the-chasm` |
| Pricing and packaging | `founder-playbook-monetizing-innovation` | `founder-playbook-100m-offers` |
| Offer, copy, page, or funnel | `design-offer-funnel` | `founder-playbook-storybrand`, `founder-playbook-made-to-stick` |
| Channel selection | `founder-playbook-traction` | `founder-playbook-100m-leads` |
| Meta planning or audit | `plan-meta-ads` | `design-offer-funnel` |
| Complex B2B sales | `founder-playbook-spin-selling` | `founder-playbook-influence` |
| MVP, experiment, pivot | `founder-playbook-lean-startup` | `validate-business-idea` |
| Business setup and delivery system | `build-business-operations` | `reason-business-decision` |
| Correction, failed advice, measured outcome, or repeated mistake | `improve-live-consultant` | the affected domain skill |

The namespaced Founder Playbook skills route to the pinned upstream snapshot.
They are reference lenses, not authority. The curated skills own synthesis,
current verification, economics, safety, and permissions.
